# VacuTech 진공장비 웹사이트 디자인 아이디어

## 아이디어 1: 산업 정밀 미학 (Industrial Precision Aesthetic)

<response>
<text>
**Design Movement**: 독일 바우하우스 + 현대 산업 디자인
**Core Principles**:
- 정밀 기계의 구조적 아름다움을 시각화
- 기술적 신뢰감을 주는 차가운 금속 팔레트
- 데이터와 스펙을 전면에 내세운 정보 중심 레이아웃
- 수직/수평 그리드를 엄격히 활용한 기계적 구성

**Color Philosophy**: 딥 네이비(#0A1628) 배경에 스틸 블루(#4A7FA5)와 크롬 실버(#C8D6E5) 강조색. 진공의 어둠과 금속 장비의 광택을 표현.

**Layout Paradigm**: 비대칭 2단 분할 — 좌측 넓은 텍스트 블록, 우측 기계 이미지. 섹션마다 수직 구분선으로 공학 도면 느낌 연출.

**Signature Elements**:
- 얇은 1px 선으로 구성된 기술 도면 스타일 장식
- 숫자 카운터 애니메이션 (압력 수치, 납품 실적 등)

**Interaction Philosophy**: 스크롤 시 수치가 카운트업되는 효과, 호버 시 제품 스펙 오버레이

**Animation**: 페이지 진입 시 좌→우 슬라이드인, 수치 카운터 애니메이션

**Typography System**: Bebas Neue (헤드라인) + Noto Sans KR (본문)
</text>

## 아이디어 2: 테크 다크 모드 (Tech Dark Mode)

<response>
<text>
**Design Movement**: 사이버펑크 + 하이테크 B2B SaaS
**Core Principles**:
- 어두운 배경 위 네온 강조색으로 첨단 기술 이미지 강조
- 글로우 효과와 그라디언트로 에너지감 표현
- 카드 기반 모듈형 레이아웃
- 미래지향적이고 혁신적인 브랜드 포지셔닝

**Color Philosophy**: 차콜 블랙(#0D0D0D) 배경, 일렉트릭 시안(#00D4FF) 강조, 딥 퍼플(#6B21A8) 보조색. 진공 기술의 첨단성을 표현.

**Layout Paradigm**: 풀스크린 히어로 + 카드 그리드. 각 카드에 미세한 글로우 테두리.

**Signature Elements**:
- 네온 글로우 테두리 카드
- 배경 파티클 애니메이션 (진공 속 분자 이미지)

**Interaction Philosophy**: 호버 시 카드가 부상하는 효과, 클릭 시 리플 효과

**Animation**: 파티클 플로팅, 글로우 펄스 애니메이션

**Typography System**: Orbitron (헤드라인) + Noto Sans KR (본문)
</text>

## 아이디어 3: 프리미엄 코퍼레이트 (Premium Corporate)

<response>
<text>
**Design Movement**: 스위스 국제 타이포그래피 + 럭셔리 B2B
**Core Principles**:
- 여백을 적극 활용한 고급스러운 레이아웃
- 오프화이트와 딥 네이비의 클래식한 대비
- 사진과 텍스트의 균형 잡힌 편집 디자인
- 신뢰와 전문성을 전달하는 절제된 미학

**Color Philosophy**: 크림 화이트(#F8F5F0) 배경, 딥 네이비(#1B2A4A) 주색, 골드 앰버(#C8922A) 포인트. 고급 산업 브랜드의 권위감.

**Layout Paradigm**: 잡지식 비대칭 편집 레이아웃. 대형 타이포그래피와 풀블리드 이미지를 교차 배치. 좌우 여백을 넓게 활용.

**Signature Elements**:
- 굵은 세리프 헤드라인과 얇은 본문의 극적 대비
- 골드 라인 구분자와 섹션 마커

**Interaction Philosophy**: 부드러운 패럴랙스 스크롤, 섬세한 호버 언더라인 효과

**Animation**: 스크롤 기반 페이드인, 이미지 줌 패럴랙스

**Typography System**: Playfair Display (헤드라인) + Noto Sans KR (본문)
</text>
<probability>0.09</probability>
</response>
